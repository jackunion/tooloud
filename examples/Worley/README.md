# Worley noise

![](/examples/Worley/img/e1.png)

```javascript
const n = tooloud.Worley.Euclidean(x, y, z);
data[index + 0] = Math.floor(255 * n[0]);
data[index + 1] = Math.floor(255 * n[0]);
data[index + 2] = Math.floor(255 * n[0]);
data[index + 3] = 255;
```
